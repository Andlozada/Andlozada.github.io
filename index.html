Mi glosario personal de Sistemas Operativos

Por: Andres Felipe Lozada



A continuación encontrarás las definiciones de las principales palabras y expresiones relacionadas con la a

Arquitectura de Red

Se refiere al diseño y la estructura de una red de computadoras, incluyendo sus componentes y cómo están interconectados



ARPANET

Fue la precursora de Internet, desarrollada por la Agencia de Proyectos de Investigación Avanzada (ARPA) de los Estados Unidos en la década de 1960



Conexión Malla

Es una topología de red en la que cada dispositivo está conectado directamente a todos los demás dispositivos, creando múltiples rutas para la comunicación



Cisco

Es una empresa multinacional especializada en tecnologías de redes de computadoras, incluyendo hardware, software y servicios



Conexión Remota

Es la capacidad de acceder y controlar un dispositivo o sistema desde otro lugar, a través de una red, como Internet



Capa de Aplicación

Es la capa del modelo OSI que proporciona servicios de red a las aplicaciones del usuario, como correo electrónico y transferencia de archivos



Capa de Presentación

Es la capa del modelo OSI que se encarga de la representación de datos, incluyendo la codificación y compresión de datos



Capa de Transporte

Es la capa del modelo OSI que proporciona un servicio de transporte confiable y transparente entre dispositivos finales



Capa de Red

Es la capa del modelo OSI que se encarga de enrutar y conmutar los datos a través de la red



Capa de Enlace de Datos

Es la capa del modelo OSI que se encarga de la transmisión de datos entre dispositivos adyacentes en una red



Capa Física

Es la capa del modelo OSI que se encarga de la transmisión de bits a través del medio físico de la red



Capa de Sesión

Es la capa del modelo OSI que establece, mantiene y finaliza sesiones de comunicación entre dispositivos



DARPA

Es la Agencia de Proyectos de Investigación Avanzada de Defensa, una agencia del Departamento de Defensa de los Estados Unidos responsable de desarrollar nuevas tecnologías para uso militar



Direccionamiento IP

Es un identificador numérico único asignado a cada dispositivo conectado a una red que utiliza el protocolo de Internet (IP)



Dominio en Correos

Es el nombre de dominio que identifica la ubicación de un servidor de correo electrónico en Internet



Ethernet

Es una tecnología de redes de computadoras que define estándares para la transmisión de datos en redes locales (LAN)



Enlace de Datos

Es la capa del modelo OSI que se encarga de establecer y mantener la comunicación entre dispositivos adyacentes en una red



HTTP

Protocolo de Transferencia de Hipertexto, es el protocolo utilizado para transferir datos en la World Wide Web



LAN

Es una Red de Área Local, que se refiere a una red de computadoras que abarca un área geográfica limitada, como un edificio o una empresa



HTTPS

Es una extensión segura del protocolo HTTP, que utiliza cifrado SSL/TLS para proteger la comunicación entre el navegador y el servidor web



HTML

Lenguaje de Marcado de Hipertexto, es el lenguaje de marcado utilizado para crear páginas web



Internet

Es una red global de computadoras interconectadas que utilizan el protocolo de Internet (IP) para comunicarse entre sí



Modelo de Referencia ISO

Se refiere al modelo de referencia de Interconexión de Sistemas Abiertos, que establece un marco para la comunicación entre sistemas informáticos



Modelo OSI

Se refiere al modelo de Interconexión de Sistemas Abiertos, que define siete capas para la comunicación de red



NCP

Protocolo de Control de Red, fue un protocolo utilizado en las primeras etapas de desarrollo de ARPANET antes de ser reemplazado por TCP/IP



Nodo

Es un punto de conexión en una red donde los dispositivos pueden enviar, recibir o reenviar datos



Protocolo

Es un conjunto de reglas y estándares que define cómo se comunican los dispositivos en una red



Protocolo TCP/IP

Es el conjunto de protocolos de comunicación utilizado en Internet para la transmisión de datos



Protocolo de Internet

Es el conjunto de protocolos que permite la comunicación en Internet, incluyendo IP, ICMP, TCP y UDP, entre otros



Red

Es un conjunto de dispositivos electrónicos, como computadoras y servidores, conectados entre sí para compartir recursos y comunicarse



Red CICLADES

Fue una red de computadoras experimental desarrollada en Francia en la década de 1970, considerada como una de las precursoras de Internet



Router

Es un dispositivo de red que dirige el tráfico de datos entre redes, determinando la mejor ruta para enviar los datos



Servidor

Es una computadora o sistema que proporciona servicios, recursos o datos a otras computadoras en una red, conocido como clientes



SMTP

Protocolo Simple de Transferencia de Correo, es un protocolo utilizado para enviar correo electrónico a través de Internet



Servidor Web

Es un programa de software que se ejecuta en un servidor y responde a las solicitudes de los clientes a través de HTTP, entregando páginas web y otros recursos



Switch

Es un dispositivo de red que conecta múltiples dispositivos en una red local y dirige el tráfico de datos solo a los dispositivos de destino adecuados



Sesión

Es la capa del modelo OSI que gestiona la apertura, mantenimiento y cierre de sesiones de comunicación entre dispositivos



Tiempo Compartido

Es un modelo de computación en el que múltiples usuarios comparten simultáneamente el uso de un mismo sistema o recurso



TCP

Significa Protocolo de Control de Transmisión, un protocolo de comunicación utilizado en Internet para enviar y recibir datos de manera confiable y ordenada



WAN

Es una Red de Área Amplia, que se refiere a una red que abarca un área geográfica más extensa, como una ciudad, país o incluso a nivel mundial
